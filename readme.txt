== Description ==

To mention the current day with the opening and closing times of physical services. Users could easily change text color, background color, weekly holiday, and custom opening hours. 

== Installing the Easy Business Hours plugin ==

* Install from the WordPress Dashboard -> Plugin page or manually download the plugin and extract to wp-content/plugins/.
* Activate the plugin.
* Go to WordPress Dashboard -> Easy Business Hours and Use the shortcode to display the output on pages or posts.
